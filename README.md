# Chrome-Extension
Chrome extension built using HTML, CSS and JavaScript to instantly save and manage active tab URLs

## Features: 
1. Uses 'chrome.tabs' API to fetch current URL.
2. Saves all links using browser's local storage.
3. Double-click deletion to clear cache.

## How to Use:
1. Clone the repository.
2. Open chrome and go to 'chrome://extensions/'.
3. Toggle on 'Developer Mode'
4. Click on "Load Unpacked" and select this project's folder.
